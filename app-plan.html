Personal Web Application Plan

WhiskedAway

WhiskedAway is a wedding catering website that presents elegant food options, clear pricing, and easy contact details for couples planning a special event.

Purpose and Audience

The purpose of this website is to help people explore WhiskedAway's wedding catering services in one polished place. The site will show menu photos, package pricing, and simple contact information so couples can quickly decide if the service fits their event.

The intended audience is engaged couples, wedding planners, and family members helping organize a wedding reception. They will likely want a website that feels professional, trustworthy, and easy to understand on a phone or computer.

Dynamic Elements

JavaScript will be used to make the site more interactive without making it too complicated. I plan to build a food gallery that can change based on category, such as appetizers, main dishes, and desserts. I also want to add a simple pricing display that updates when a user chooses a guest count or package option.

Another dynamic feature will be a contact section that can show a helpful message after a form is submitted. These features will make the site feel more modern while still being simple enough for a beginner to code.

Pseudocode

  1. Load the page and display the WhiskedAway heading, menu photos, pricing list, and contact information.
  2. Store the catering packages and food items in simple arrays or objects.
  3. When a user clicks a food category, show only the matching food cards.
  4. When a user changes the guest count, update the estimated price.
  5. When the contact form is submitted, prevent the page from refreshing and show a friendly confirmation message.
  6. Keep the layout responsive so the site still looks clean on smaller screens.

Site Content

  • Images: elegant photos of appetizers, plated meals, desserts, and event tables.
  • Pricing: simple package levels with per-guest estimates and add-on options.
  • Contact: phone number, email address, and a short request form for quotes.
  • Style: soft colors, clean spacing, professional typography, and a luxury wedding feel.
